Find all papers citing X and Y

This tool allows you to find out if two research branches ever crossed. Have you ever wondered if someone applied neural networks to detect wildlife before? With this tool, you can find it out: Paste some highly cited neural network papers (Adam, YOLO, Feature Transfer) and different wildlife papers (Camera trap, lions, hyenas, and cheetahs, another wildlife camera trap).

If you use the tool well, you might be able to write "We are the first to applied X to Y." and delete the "To the best of our knowledge," phrase.



The tool is built on the Semantic Scholar API. Created by Leon Sixt.
